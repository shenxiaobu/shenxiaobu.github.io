一步一步完成个vue项目

环境： git node

  1. 安装vue脚手架

跳出版本号 即表示安装成功

  1. 创建一个项目 vue-study

-d 表示按默认配置创建一个项目

  1. 执行vue ui 使用可视化界面控制项目并安装所需依赖

浏览器就会跳出项目可视化界面，可自行导入项目

点击依赖展开的界面就可以查看项目已安装的依赖，未安装的依赖也可以在这进行安装

安装了vue全家桶 vuex、vue-router、axios、qs

点击运行既可以将项目跑起来也可以执行命令

点击输出 就能看到本地链接，点击链接就能看到项目

  1. 在vscode编辑器上打开该文件，目录展示如下

可见目录与我们想要的目录还是不够的

  1. 将目录建成如下结构

src //源代码

api //所有请求

assets //主题 字体等静态资源

components //全局公用组件

directive //全局指令

filters //全局过滤器

icons //项目所有svg icons

lang //国际化 language 非必须

mock //项目mock模拟数据 非必须

router //路由

store //全局store 管理

styles //全局样式

utils //全局公用方法

vendor //公用vendor 非必须

views //view

App.vue //入口页面

main.js //入口加载组件初始化

permission.js //权限管理 非必须

static

Tinymce //富文本

public

favicon.ico //favicon图标

index.html //html模板

babel.config.js //babel-loader 配置

eslintrc.js //eslint配置项

.gitignore //git忽略项

package.json //package.json

README.md //使用说明书

vue.config.js //配置文件
