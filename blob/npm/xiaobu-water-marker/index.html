# xiaobu-water-marker的使用

# 安装

#install by yarn
yarn add xiaobu-water-marker

#install by npm
npm install xiaobu-water-marker

# 引入

import { XiaobuWaterMarker } from "xiaobu-water-marker"
Vue.use(XiaobuWaterMarker)
或者

const XiaobuWaterMarker = require("xiaobu-water-marker")
Vue.use(XiaobuWaterMarker)

# 使用

# 水印文字

<xiaobu-water-marker text="爆米花小布" />

# 配置参数

字段 说明 类型 默认值
text 文字水印文本 String 空（必传）
text-color 文本颜色 String rgb(180, 180, 180)(同css的Color属性可取值)
canvas-font 字体大小 String 16px Source Han Sans
width 文本容器宽度 Number 124
height 文本容器高度 Number 124
z-index 文本容器的层级 Number 9
opacity 文本容器的透明度 String 0.3
rotate-deg 文字旋转的角度 Number 20
rotate-direction 文字旋转的方向 String clockwise 顺时针 anti-clockwise 逆时针（默认）
crisscross 水印文字是否隔行交错 Boolean false
margin-left 水印交错的距离 Number 文本宽度的一半（传入负数时会取绝对值）
odd-or-even 奇数行或偶数行做左边距的交错 Number 0 奇数 1 偶数（默认）
text-padding 文本左右内边距（解决当文本旋转一定角度时文本显示不全的问题） Number 0 传入负数时会取绝对值

# 水印图片

<xiaobu-water-marker :imgSrc="require('./imgs/12blue.jpg')" />

# 配置参数

字段 说明 类型 默认值
width 文本容器宽度 Number 124
height 文本容器高度 Number 124
z-index 文本容器的层级 Number 9
opacity 文本容器的透明度 String 0.3
rotate-deg 文字旋转的角度 Number 20
rotate-direction 文字旋转的方向 String clockwise 顺时针 anti-clockwise 逆时针（默认）
crisscross 水印文字是否隔行交错 Boolean false
margin-left 水印交错的距离 Number 文本宽度的一半（传入负数时会取绝对值）
odd-or-even 奇数行或偶数行做左边距的交错 Number 0 奇数 1 偶数（默认）
left-instance 图片左边内间距 Number 20
right-instance 图片右边内间距 Number 20
top-instance 图片上边内间距 Number 20
bottom-instance 图片下边内间距 Number 20
gray-img 是否将图片置灰 Boolean true
rgb-num 置灰系数 Array [0.299, 0.587, 0.114]
img-color 图片颜色 String 无默认值 十六进制逗号隔开 如 "255,0,125"
<xiaobu-water-marker 
  :imgSrc="require('./imgs/12blue.jpg')" 
  :width="66" 
  :height="66" 
  :z-index="1" 
  opacity="1" 
  :rotate-deg="10" 
  rotate-direction="clockwise" 
  :crisscross="true" 
  :margin-left="-20" 
  :odd-or-even="0" 
  :left-instance="50" 
  :right-instance="50" 
  :top-instance="50" 
  :bottom-instance="100" 
  :rgb-num="[1,1,1]" 
  :gray-img="false" 
  img-color="0,0,255" 
/>

img-color gray-img rgb-num 同时存在时 img-color有效 其余两个无效

rgb-num 当且仅当 gray-img 为true 且未传入 img-color 时 有效

  <div class="img-box">
      <img class="img" :src="require('./imgs/12blue.jpg')" alt="">
      <xiaobu-water-marker 
        :showCanvas="true" 
        :openGuide="true" 
        :width="100" 
        text-baseline="bottom"  
        textAlign="right" 
        :waterText="true" 
        :height="100" 
        text-color="#fff" 
        opacity="1" 
        :rotateDeg="0" 
        :isParentStatic="false" 
        text="小小小布"
      />
  </div>
字段 说明 类型 默认值
showCanvas 是否展示canvas元素 Boolean false
waterText 是否水印图片 Boolean false
centerX 水印文字x轴距离 Number -
centerY 水印文字Y轴距离 Number -
openGuide 是否打开参考线 Boolean false
guideColor 参考线颜色 String #eee
